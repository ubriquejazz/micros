# Stmas en Tiempo Real

SETR - Universidad de Alcala de Henares (2025)
[profesor](https://www.uah.es/es/estudios/profesor/Jose-Luis-Martin-Sanchez/)

This course deals with the problem of the design of real-time electronic systems, first
considering the model of tasks and the analysis of executability, to continue with the
techniques for designing applications for embedded systems

Se sigue con el estudio de RTOS estricto para sistemas empotrados. Por último, se completa la asignatura con el estudio de Linux como sistema operativo. 

El enfoque de la asignatura será práctico, desarrollando desde el principio una aplicación que deberá ejecutarse en FreeRTOS
