# STM32F411 Discovery kit

Tarjeta de desarrollo con programador y depurador integrado: STM32F411 Discovery kit (32F411EDISCOVERY)

## Peripherals

a gyroscope, an e-compass, an OTG Micro-AB connector
a digital microphone, an audio DAC with integrated speaker driver

|  Outputs |  GPIO |
|-------|-------|
| LD3 (orange) | PD13 |
| LD5 (red) | PD14 |
| LD4 (green) | PD12 |
| LD6 (blue) | PD15 |

|  Inputs |  GPIO |
|-------|-------|
| Button (blue) | PA0

## Programmer

To ST-LINK/V2 embedded debug tool and ST-LINK server

## Real time clock (RCC)

- Low Speed Clock: Bypass clock source
- Change HSE from "BYPASS clock source" to "Crystal/Ceramic Resonator"
